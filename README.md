## 🚀 Project Overview

SmartEvent is a comprehensive web application designed for event organizers to manage events and track real-time occupancy. The project is built with a modular Flask backend and a dynamic SQL Server database.

## 🛠️ Key Features

- **Secure Authentication:** Robust login/register system with session management.
- **Dynamic Dashboard:** Real-time overview of all active events.
- **Occupancy Analysis:** Visual progress bars showing live ticket sales data.
- **Interactive UI:** Seamless ticket purchasing using JavaScript Fetch API without page reloads.

## 💻 Tech Stack

- **Backend:** Python (Flask)
- **Database:** MS SQL Server
- **Frontend:** HTML5, CSS3 (Premium UI), JavaScript (Vanilla)
- **Architecture:** Modular Pattern (Auth, Events, Tickets)

## 🔧 Installation

1. Clone the repository: `git clone https://github.com/bngssntrk5/Smart-Event-Ticket-System.git`
2. Install dependencies: `pip install flask pyodbc python-dotenv`
3. Configure your `.env` file with your SQL Server credentials.
4. Run the app: `python app.py`
